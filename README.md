# Primary Suite Remodel — 3D Mock-up

An interactive, to-scale 3D model of a primary bathroom and vanity remodel, built to share with contractors.

Open `index.html` in any browser, or view the hosted page via GitHub Pages.

**Controls**
- Drag to orbit, scroll to zoom, right-drag to pan.
- **Current / Proposed** — toggle between the existing layout and the remodel.
- **Display** — Labeled, Clean (no text), or Floor plan (top-down architect view).
- **Bathroom vanity** — South ·46″ or North ·~72″ (Option A) layout.
- Camera presets, plus Ceilings and Furniture toggles.

Dimensions and finishes are a work in progress — see the "Assumptions to verify" panel in the page.
